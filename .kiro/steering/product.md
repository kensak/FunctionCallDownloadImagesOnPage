# Product Overview

更新日: 2026-01-18

指定したWebページから画像（`<img src="...">`）を抽出し、ローカルに一括ダウンロードする Python のCLIツールです。
開発者・利用者が「ページ上の画像をまとめて取得して整理したい」場面を、最小限の手順で支援します。

## Core Capabilities

- **静的HTMLからの画像抽出**: HTMLを取得して `img` 要素の `src` を抽出し、絶対URLに解決
- **動的ページ対応（オプション）**: `--playwright` で JavaScript レンダリング後の画像をキャプチャして保存
- **サイズフィルタ**: `--min-width/--min-height` および `--max-width/--max-height` による除外
- **安全な保存**: 同名ファイル衝突時に連番サフィックスでユニーク名を生成して上書きを回避
- **堅牢な実行**: 画像単位の失敗は警告ログとして扱い、処理を継続（致命的な取得/書き込みは終了コードで通知）

## Target Use Cases

- 記事ページやドキュメントページの画像をローカルにまとめて保存したい
- 静的サイトの画像収集（簡易スクレイピング）
- 画像が多いページを対象に、最小/最大サイズで必要な画像だけ抽出したい
- SPA 等の動的ページで、レンダリング後に表示される画像を取得したい

## Value Proposition

- **CLIとしてすぐ使える**（パッケージの `project.scripts` でコマンド提供、`python -m` も後方互換として維持）
- **責務分離された構成**（引数解析・取得・解析・保存・フィルタ・例外を分け、変更に強い）
- **失敗に強い設計**（1枚の失敗で全体を止めない、要約ログで結果を把握しやすい）
